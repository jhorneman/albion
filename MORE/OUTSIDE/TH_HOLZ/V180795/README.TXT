Kurzeinfhrung - readme:

Um ein Readme zu machen kann man
dieses File als Template verwenden.
Natrlich kann man auch ein ganz
normales Textfile zum readme er-
nennen. Dann sollte man aber zu-
mindest darauf achten, daá ma nicht
breiter als 36 Zeichen wird. (Wir
wollen schlieálich nicht, daá der
Benutzer dauernd scrollen muá,
oder ?)

Zur Formatierung von Readme's kann
der Downslash verwenden werden.
Den benutze ich zum Beispiel um die
Seitenangabe dort hin zu bekommen:
